Instructivo para el uso de la plantilla

Cuando abra Microsoft Word Word, seleccione “Archivo Nuevo”, y en lugar de usar un documento en blanco, elija la plantilla llamada “autores_digital”.

2.1

Barra de herramientas "Materiales didácticos"

Texto

La plantilla tiene una barra de herramientas con tres menúes desplegables, su función es proveer el diseño predeterminado para distintos requerimientos:

  1. Menú “recursos didácticos”: inserta en el documento el diseño predeterminado para redactar alguno de los 4 recursos disponibles.
    Ejemplo:
  2. Menú “recursos multimedia”: genera en el documento el diseño para insertar una imagen o crea el ícono destinado a linkear distintos recursos multimedia disponibles en una dirección de Internet.
    Ejemplo:
  3. Menú “Referencias Bibliográficas”: genera los recursos para automatizar la inserción de cada tipo de cita bibliográfica.
    Ejemplo:

Cómo hacer visible la barra de herramientas:
Si la barra de herramientas no está visible, Abra el menú “Ver”, “Barra de herramientas”: en esta opción se desplegará una lista de barras, active la que se llama “Materiales didácticos”.

2.1.1

En Word 2007

Texto

Una vez que haya creado el documento nuevo basado en la plantilla, verifique si se muestran las barras de herramientas para insertar los recursos didácticos.
Si no están visibles, habilítelas desde el menú llamado “Complementos”

2.2

Acerca de los Estilos

Texto

Además de las herramientas de diseño predeterminado, la plantilla provee distintos estilos para dar formato a los títulos y párrafos.

Estos estilos pueden seleccionarse desde la lista desplegable que aparece junto a las demás opciones de formato de párrafos, como muestra la imagen derecha.

El estilo de texto predeterminado se llama “UVQ Normal”, y es el que funciona por defecto, a menos que cambie el formato reemplazando por los demás estilos disponibles.

Estos estilos de párrafo predeterminados pueden usarse de dos modos diferentes:

  1. Insertar el estilo deseado en un punto determinado de la página y luego escribir allí.
  2. Copiar texto escrito previamente en otro documento y pegarlo en un punto de inserción al que previamente se le asignó el estilo deseado.

Como podemos observar en el segundo modo, no es imprescindible que los contenidos se escriban directamente en el documento de plantilla: puede recuperar el texto ya escrito y pegarlo en el nuevo documento basado en esta plantilla, pero para esto es muy importante usar el método de “pegado especial... texto sin formato”, en lugar del sistema habitual de pegado.

2.2.1

Para usar pegado especial

Texto
  1. Seleccione el texto en el documento fuente y cópielo mediante el comando “Edición... Copiar”, o “Ctrl. + C”
  2. En el documento de destino ubique el cursor donde desea incluir el /los párrafos copiados y cliquee el menú Edición... Pegado Especial y seleccione la opción “texto sin formato”.

Estos dos modos de edición también son válidos para escribir o pegar contenidos dentro de las tablas de recursos didácticos y lecturas.
Asimismo, para el segundo modo debe usarse el método de “pegado especial... texto sin formato”.

2.2.2

Jerarquía de títulos

Texto

La taxonomía para títulos establece el siguiente orden:
____UVQ Título de Unidad (nivel 1.)
___UVQ Título de Sección (nivel 1.1.)
__UVQ Título de Apartado (nivel 1.1.1.)
_UVQ Subtítulos (nivel interno de apartados, sin numeración)

Use cada uno de estos niveles para establecer la correcta jerarquía de contenidos: recuerde que éstos se verán reflejados en el índice.

Ejemplo

Cómo asignar estilo al título

  • escribir el título
  • seleccionar el título pintando con el mouse
  • desplegar el menú de estilos y seleccionar
    “UVQ título de apartado”, por ejemplo.
2.3

Uso de la Barra “Referencias Bibliográficas”

Texto

Esta herramienta se utiliza para citar correctamente las distintas Referencias Bibliográficas
La forma de utilizarla es la siguiente:

  1. Ubique el punto de inserción donde deba comenzar la referencia
  2. Haga un clic en el ícono de la referencia deseada, desde la barra de referencias. Automáticamente se generará un texto con el formato recomendado para esa referencia, y quedará seleccionado el primer campo.
  3. Reemplace el texto seleccionado por el dato correcto.
  4. Haga un clic sobre el segundo término de la referencia y repita la operación de reemplazar por el texto correcto. Repita este paso con cada uno de los términos de la referencia. 

Para agregar otra referencia coloque el punto de inserción debajo de la última. Los estilos de referencias tienen un modo especial de selección de texto para facilitar la tarea: al hacer un clic sobre el término, se selecciona todo el texto demo para reemplazarlo por los datos correctos. Una vez reemplazado el texto, pierde este atributo.
